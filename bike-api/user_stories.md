# User Stories for EiffelBike Rental and Sales System


* **US_01:** As a Student, I want to offer my bike for rent so that other university members can rent it.

* **US_02:** As an Employee, I want to offer my bike for rent so that other university members can rent it.

* **US_03:** As EiffelBikeCorp, I want to offer company bikes for rent so that customers can rent bikes even when no private bikes are available.

* **US_04:** As EiffelBikeCorp, I want to offer bikes for sale with detailed notes so that buyers can assess the bike’s condition before purchasing.

* **US_05:** As a Customer, I want to search for bikes available to rent so that I can find a suitable bike quickly.

* **US_06:** As a Customer, I want to rent a bike so that I can use it for my commute or daily needs.

* **US_07:** As a Customer, I want to pay the rental in any currency and have it converted to euros so that I can pay with my preferred currency while the system remains consistent.

* **US_08:** As a Customer, I want to add notes when returning a bike so that the next renter and the bikeProvider know the bike’s condition.

* **US_09:** As a Customer, I want to be added to a waiting list when a bike is not available so that I can rent it as soon as it becomes available.

* **US_10:** As a Customer, I want to receive a notification when a bike becomes available so that I can rent it before someone else does.

* **US_11:** As a Customer, I want to search for bikes available to buy so that I can find a bike to purchase.

* **US_12:** As a Customer, I want to view the notes associated with bikes offered for sale so that I can make an informed buying decision.
